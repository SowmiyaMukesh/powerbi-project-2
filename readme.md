# Project Overview

In this project, I worked on a retail grocery dataset and built an interactive Power BI dashboard to analyze sales performance across different outlet types, locations, and product categories.
The goal was to understand how various factors such as outlet size, location, and item type impact overall sales and performance.
I used Power BI to clean, structure, and visualize the data, creating a dashboard that provides a clear view of key business metrics.
________________________________________
# Dataset

The dataset contains grocery retail data including:
•	Sales information
•	Item details (type, fat content, visibility)
•	Outlet details (location, size, type)
•	Ratings and performance indicators
________________________________________
# Data Preparation

In Power BI, I performed:
•	Data cleaning and formatting
•	Handling categorical fields (item type, outlet type, etc.)
•	Structuring the dataset for analysis
•	Creating calculated metrics for KPIs

# Dashboard Features

The dashboard includes:
•	Sales trend analysis based on outlet establishment year
•	Sales comparison across outlet sizes and locations
•	Item category performance analysis
•	Fat content distribution analysis
•	Outlet-type performance comparison
Slicers are used to filter the data dynamically based on outlet location type, outlet size, and item type.
________________________________________
# Visualizations Used

•	KPI Cards
•	Line Chart (trend analysis)
•	Donut Charts (distribution)
•	Bar Charts (category comparison)
•	Table (detailed performance view)
•	Slicers for interactivity

# Key Insights

•	Sales performance varies across different outlet sizes and location tiers
•	Certain item categories contribute more significantly to total sales
•	Outlet type plays a key role in sales distribution
•	Product attributes such as fat content and visibility influence performance
